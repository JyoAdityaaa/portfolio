01 / 03
Competitor AI
LLM-powered competitive intelligence, automated.

Input a product name and niche — Competitor AI auto-discovers relevant rivals, runs feature-by-feature LLM analysis on each, and returns structured comparisons with positioning insights. No manual research required.

Problem
Competitive research is slow, inconsistent, and hard to scale for early-stage teams.
Approach
Python + Web Search APIs to discover rivals. LLM pipeline to extract and compare features at scale.
Outcome
Structured competitor cards with positioning gaps — actionable in minutes, not hours.
What I Learned

LLMs hallucinate competitor facts without grounding. Added a web-search verification layer before any LLM pass — essential for reliable output.

02 / 03
AI SWOT Analyzer
Gemini · Google Sheets · PDF — one pipeline.

Input a company or product. Gemini API generates a structured SWOT, writes it live to Google Sheets for collaboration, and exports a formatted PDF — all in one pass.

Problem
SWOT analyses are manual, inconsistently formatted, and trapped in silos.
Approach
Gemini API for generation. Google Sheets API for live output. PDF automation for clean exports.
Outcome
Single-click pipeline from input to shareable SWOT doc with consistent formatting every time.
What I Learned

Sheets API rate limits hit hard at scale. Batching writes and adding retry logic was essential for reliability.

03 / 03
AQI Visualizer
Air quality data for regions that have none.

Built for underserved rural regions with zero existing AQI infrastructure. Pulls from public APIs and renders interactive dashboards — giving communities visibility into something that directly affects their health.

Problem
Millions in rural India have no accessible AQI data despite severe pollution.
Approach
Open AQI APIs + Streamlit for rapid interactive UI. Python pipeline for processing and rendering.
Outcome
Interactive regional AQI dashboard deployable for areas with no dedicated monitoring.
What I Learned

Open AQI APIs have massive gaps for rural areas. Built an interpolation layer to estimate values for uncovered coordinates.

Lab Notes

Lab notes.

Short notes on what I’m learning, building, and figuring out.

Feb 2026
Structured LLM outputs are a product decision, not just a technical one

While building Competitor AI, I found that asking an LLM for “a comparison” produced inconsistent lengths and formats. Enforcing JSON schema in the prompt — and validating before rendering — fixed 80% of the quality issues. The prompt is the product spec.

LLMsPrompt Engineering
Jan 2026
API chaining is where AI projects actually break

Integrating Gemini API + Google Sheets API + PDF export in one pipeline taught me that error handling at each layer is non-negotiable. A silent failure in the Sheets auth broke the whole chain. Now I always log and validate at every API boundary.

IntegrationPythonAPIs
Dec 2025
Agentic AI is mostly about trust, not capability

At GDG Agentathon, the hardest design question wasn’t “can the agent do this?” but “when should it ask for confirmation?” Users don’t trust agents that act silently. Human-in-the-loop checkpoints are a feature, not a fallback.

Agentic AIUXGDG Hackathon
Jun 2025
Streamlit is a legitimate prototyping superpower

For the AQI Visualizer I went from idea to deployed app in two days with Streamlit. For internal tools and AI demos, it removes all the frontend friction and lets you test the actual product hypothesis. Not production-forever, but perfect for signal-finding.
